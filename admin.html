Needs Contents: Read & Write permission on this repo (covers products.json edits and Release image uploads). How do I create one?

Usually main. Change if your default branch is named differently.

How to create your GitHub token

  1. Go to GitHub Settings → Developer settings → Personal access tokens → Fine-grained tokens and click Generate new token.
  2. Token name: anything (e.g. fareed-studio admin).
  3. Expiration: choose 90 days or longer (you can renew anytime).
  4. Repository access: choose "Only select repositories" → pick your fareed-studio repo.
  5. Permissions: expand Repository permissions, find Contents, and set it to Read and write.
    That single permission covers everything this panel needs: editing products.json, creating the Releases tag, and uploading image assets. No other permission toggles are required.
  6. Click Generate token, copy the github_pat_... string, and paste it in the field above.
  7. The token only gives access to this one repo.
